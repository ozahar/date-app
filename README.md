# デートアプリ

全国47都道府県のデートスポットを検索・表示するWebアプリケーションです。

## 機能

- **県別検索**: 47都道府県からスポットを検索
- **ジャンル別検索**: 絶景、グルメ、歴史、観光、アウトドアなどのジャンルでフィルタリング
- **予算別検索**: 無料・有料で絞り込み
- **地図表示**: Leaflet.jsを使用したインタラクティブな地図表示
- **スポット詳細**: 各スポットの写真、説明、タグ、予算情報を表示

## 技術スタック

- HTML5
- CSS3
- Vanilla JavaScript
- Leaflet.js (地図表示)
- CartoDB (地図タイル)

## ローカルでの実行

1. リポジトリをクローン
```bash
git clone <repository-url>
cd date-app
```

2. index.htmlをブラウザで開く

## デプロイ

このアプリは静的サイトとしてVercelにデプロイできます。

### Vercelデプロイ手順

1. GitHubにリポジトリをプッシュ
2. Vercelダッシュボードで「New Project」をクリック
3. GitHubリポジトリをインポート
4. デプロイ設定を確認して「Deploy」をクリック

## プロジェクト構成

```
date-app/
├── index.html      # メインHTMLファイル
├── script.js       # アプリケーションロジック
├── style.css       # スタイルシート
├── data.js         # スポットデータ
├── vercel.json     # Vercel設定ファイル
├── .gitignore      # Git無視ファイル
└── README.md       # このファイル
```

## データについて

- 全47都道府県に3〜5つの代表スポットを収録
- 各スポットには正確な緯度・経度座標を指定
- 写真映えやドライブに適した高プロフィールなスポットを選定
